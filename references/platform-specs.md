# Social Media Platform Specs

Reference for the `social-media` skill. Per-platform limits, conventions, and tone presets.

## Platform specs

### Facebook
- **Length:** No hard limit; aim for 40–80 chars for engagement (longer is fine if it earns the length).
- **Tone:** Conversational, community-minded.
- **Format:** 1–2 sentences + 1–2 emoji. End with a question to drive comments.
- **Hashtags:** 0–3, optional. Facebook hashtags barely help reach.

### Instagram
- **Caption limit:** 2200 chars.
- **Feed truncation:** After ~125 chars (first line must hook).
- **Hashtags:** 10–15. Mix broad (#ai, #programming) and niche (#sglang, #fastapi). Place at the end, separated by a blank line.
- **Tone by preset:**
  - professional: polished, brand-voice, no slang.
  - casual: friendly, first-person, light emoji.
  - promotional: hook → value → CTA → urgency.
  - educational: "Did you know…" → teach one thing → takeaway.

### YouTube
- **Title limit:** 100 chars (≤ 70 ideal for search + thumbnail).
- **Description limit:** 5000 chars.
- **Description format:** 2–3 sentence summary → timestamps placeholder (`0:00 Intro` etc.) → links section → CTA.
- **Tags:** 8–12 comma-separated. Front-load the most relevant.
- **Title conventions:** Front-load keywords; curiosity gap OK; no clickbait ("YOU WON'T BELIEVE…").

### X / Twitter
- **Limit:** 280 chars (or 4000 for X Premium — assume 280).
- **Format:** One idea per tweet. If the idea is big, offer a thread (tweet 1 = hook + "🧵").
- **Tone:** Punchy, confident, no hedging.
- **Hashtags:** 0–2. Too many hurt reach.

### LinkedIn
- **Limit:** 3000 chars.
- **Format:** Hook line (1 sentence) → blank line → 3–5 short paragraphs OR a bulleted list → end with a question.
- **Tone:** Professional, first-person, reflective. Avoid humble-brags.
- **Hashtags:** 3–5 max.

### Gmail
- **Subject:** ≤ 60 chars. Front-load the value ("Free 30-min code review" not "Quick question").
- **Body format:** Greeting (use name if known) → 2–3 short paragraphs → single clear CTA → signature placeholder.
- **Tone:** Match the recipient (cold outreach = professional + concise; newsletter = educational + warm).

## Tone presets (cross-platform)

| Preset | One-liner |
|---|---|
| professional | Polished, brand-voice, no slang, no exclamation overload. |
| casual | First-person, friendly, light emoji, conversational. |
| promotional | Hook → value → CTA → urgency. Leads with benefit, not feature. |
| educational | "Did you know…" → teach one concrete thing → takeaway. |

## Hashtag rules
- IG: 10–15, broad + niche mix, at the end.
- YouTube: 8–12 tags (not hashtags in description — actual tags field).
- X/LinkedIn: 0–5 max.
- Facebook: 0–3.
- Avoid banned/shadowbanned tags (e.g. `#happythanksgiving`, `#costumes` historically) — when in doubt, skip.
