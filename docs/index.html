The five skills

Each ships as a CodeBuddy skill (SKILL.md) and a slash command.

📊

portfolio-audit

Score your top GitHub repos against a 10-point quality bar and get a concrete fix plan (README, license, topics, Pages…).

/toolkit-skills:audit-portfolio
💼

job-application

Turn a JD + your resume into a cover letter, matched/missing keywords, interview questions, and a 7-day prep plan.

/toolkit-skills:apply
🔀

pr-draft

Paste a git diff → get a Conventional Commit message, PR description, and a changelog entry.

/toolkit-skills:pr-draft
🤖

ai-infra-helper

Navigate SGLang / vLLM-style repos: where models live, what tests to run, and fork-PR CI gotchas.

/toolkit-skills:ai-infra-helper
📣

social-media

One idea → posts for Facebook, Instagram, YouTube, X/Twitter, LinkedIn, and Gmail — with hashtags.

/toolkit-skills:social
✅

Install the plugin

CodeBuddy Code loads plugins from ~/.codebuddy/plugins/cache/local/toolkit-skills/.

# 1. Clone git clone https://github.com/theraihanrakibb/toolkit-skills.git cd toolkit-skills # 2. Symlink / copy into the local plugin cache bash scripts/sync-to-cache.sh # 3. Enable in ~/.codebuddy/settings.json # add: "toolkit-skills@local": true # 4. Restart CodeBuddy Code — skills & commands are now available

Prefer the web app instead? The repo also ships a FastAPI + React site (website/) with Docker and a CI pipeline.
